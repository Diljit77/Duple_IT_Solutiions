# Dupple IT solutions - Backend Assemenent task
## Introduction
- In this task i create a backend Build a backend service where users can create teams,
manage members, and handle a simple task board with
columns: TODO → DOING → DONE. with background job 

## tech Stack
- ExpresJS - for api , routing ,server
- mongoDB - database 
- redis - for cache memeory
- worker - for background job or task
- express-rate-limit - for rate limiting

## how to run locally
- clone the repo :
- set the env variable with the help of .env-example file.
- open terminal
- run command `npm install`
- run the redis locally 
- run server :
`npm run dev`
- in another terminal run the worker-
`npm run worker`
- run the endpoint or test it in postman with the help of postman
